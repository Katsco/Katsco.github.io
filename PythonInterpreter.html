Python Interpreter
< / >

This course project required us to develop an interpreter for a subset of the SCL language
using a programming language of our choice, including a scanner, parser, and executer,
to process input SCL programs, generate tokens, build a parse tree, execute program actions,
handle memory, and showcase successful execution of specified statements from different input files.

As the team lead, I spearheaded a dynamic group of talented individuals who shared a passion for technology and a hobby of innovation.

With this group, we successfully designed and implemented a scanner, parser, and executer in Python using a subset of the SCL language.
